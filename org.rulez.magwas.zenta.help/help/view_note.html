Adding a Note

Notes can be added to the View to allow for visual annotations. A note has no semantic meaning.

Using a Note in a View

A Note's text, font and fill colour can be edited in the Properties Window.

Double-clicking the Note opens the Properties Window, clicking on a selected Note allows you to directly edit the Note's text.

 

Adding a Note Connection

You can draw a connecting line from and to a Note using the "Note Connection" line tool in the Palette. It has no semantic meaning like the other relationship types. Double-clicking the Note Connection opens the Properties Window.
